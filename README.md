## Projects

### Simple Calculator
A basic arithmetic calculator demonstrating fundamental JavaScript operations and DOM manipulation. Features addition, subtraction, multiplication, and division with a clean, functional interface.

**Key elements:** Event listeners, conditional logic, DOM updates, input validation

### Pizza Menu Order
An interactive order form that calculates pizza prices based on size and toppings selected. Demonstrates dynamic pricing logic and user input handling.

**Key elements:** Form inputs, checkbox/radio button handling, price calculation functions, real-time display updates

### Tic Tac Toe (Produce Version)
A playful twist on the classic game featuring produce-themed markers. Includes win condition checking and game state management.

**Key elements:** Array manipulation, game logic algorithms, player turn management, win/draw detection, DOM grid interaction

### To-Do App
A task management application allowing users to add, complete, and remove items from their list. Demonstrates practical list manipulation and persistent state handling.

**Key elements:** Array methods (push, splice, filter), dynamic list rendering, event delegation, toggle functionality
